git使用

发表于 2019-07-28  |  2020-06-28   |  
  1. clone项目没有权限

解决：

采用https克隆方式，加上用户名和生成一个访问令牌

git clone https://username:AccessTokens@git.uinnova.com/doiot/proxima/proxima-menu-layer-lib.git

  1. 新建远程分支

新建一个本地分支，例如： git checkout -b feat-theme
提交commit，推到远程分支git push origin feat-theme 注意要同名，这样远程分支上就有一个feat-theme的分支了

  1. 在不同分支上推同一文件

从feat-base分支签出文件
git checkout feat-base src/core/initData.js

git status .缓存

git commit -m "feat: " 提交
