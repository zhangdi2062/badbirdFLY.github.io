介绍数组的使用方法

1. 数组定义

数组即有序的元素序列，用数组下标区分数组中的元素

ECMAScript数组与其他语言不同的是，ECMAScript数组的每一项可以保存任何类型的数据，而且数组的大小可以动态调整，随数据的增加自动增长

3. 检测数组

  1. 引用数据类型判断：instanceof

这种方法适用于一个网页或者一个全局作用域而言，网页中存在多个框架实际上存在多个全局变量，从而存在两个以上不同版本的Array构造函数

  1. Array.isArray()方法,解决上述问题
  1. 使用toString()检测对象类型

可以通过toString() 来获取每个对象的类型。为了每个对象都能通过 Object.prototype.toString() 来检测，需要以 Function.prototype.call() 或者 Function.prototype.apply() 的形式来调用，传递要检查的对象作为第一个参数，称为thisArg (来自MDN)

使用 Object.prototype.toString 配合闭包，通过传入不同的判断类型来返回不同的判断函数

4. 操作方法，添加/删除元素

  1. push(...items) 从结尾添加元素，返回数组长度
  2. unshift(...items) 从开头添加元素，返回数组长度
  3. pop() 从结尾删除元素，返回被删元素
  4. shift() 从开头删除元素，返回被删元素
  5. slice(start, end) start必需，如果是负数，则从数组尾部开始算起，-1即倒数第一个元素
    end选填，不填默认到最后一个元素
    不改变原数组，返回一个新数组，包括start，不包括end
  6. splice(start, deleteCount, ...items) 从索引start开始，删除deleteCount个元素，并添加items个元素
    start必需，如果是负数，则从数组尾部开始算起
    deleteCount必需，删除几个元素，大于等于0
    返回删除的元素数组，改变原数组
  7. concat(...items) items可以是数组形式或者单个形式
  8. fill(value, start, end) 填充元素，改变原数组
    value 填充的值
    start 元素填充的起始位置
    end 停止填充的索引位置
  9. copyWithin(targer, start, end) 覆盖元素，改变原数组
    target 必需，从索引位置开始准备复制
    start 可选，元素复制的起始位置
    end 可选，停止复制的索引位置

5. 查询方法

  1. indexOf/lastIndexOf(item, pos) 返回索引值，找不到返回-1
  2. includes 返回true/false

6. 迭代方法(每个数组项进行操作)：不改变原数组

  1. forEach() 遍历数组，对数组进行操作，没有返回值
    注意： forEach对于empty是不执行的
  1. map() 对整个数组进行操作，返回什么值，当前这个索引值就是什么值
  1. filter()筛选返回符合条件的数组
  1. every() 所有都符合条件返回true，否则返回false
  1. some() 只要有符合条件的就返回true，都不符合返回false

  1. find() 返回数组中第一个满足所提供测试函数的元素的值，用的较少

  2. findIndex() 返回数组中满足提供的测试函数的第一个元素的索引，用的较少

7.

sort() 排序 默认排序顺序是在将元素转换为字符串，然后比较它们的UTF-16代码单元值序列时构建的

[1,10,30,4].sort((a, b) => a - b) 从小到大排序，升序

返回值大于0 即a-b > 0 ， a 和 b 交换位置
返回值大于0 即a-b < 0 ， a 和 b 位置不变
返回值等于0 即a-b = 0 ， a 和 b 位置不变
