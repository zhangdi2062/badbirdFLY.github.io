防抖

释义：指触发事件后n秒再执行函数，如果在n秒内又触发了，则重新计算时间

用途：input请求数据

节流

释义：就是指连续触发事件但是在 n 秒中只执行一次函数。 节流会稀释函数的执行频率

用途：drag、mousedown、resize等

JSON.stringify的基本用法是将对象转成json字符串进行传递，即JSON.stringify(data)

  3. NaN Infinity等在对象或数组中都返回null

  4. toJSON()

除了对象参数还有两个参数作用也很大

用法

JSON.stringify(value[, replacer[, space]])

参数含义：

  1. value：必选，要转换的值（包括所有的数据类型，通常是对象或者数组）

  2. replacer：可选，可以是函数或数组，这个参数用处很大啊，前提只支持一层，多层的数据会丢失举例：

  3. space：可选，控制字符串中的间距，可以是数字或字符串，数字最大是10，可用在编辑器中

  • >> 右移运算符

5 >> 2将5转换成二进制32位101，向右移两位001，得1

可以这样计算 整数 / 2的n次幂

应用 取整

5.12 >> 0 // 5 只在数字小于2^32-1的时候靠谱

new Function() 新建函数

let func = new Function ([arg1[, arg2[, ...argN]],] functionBody)

函数的参数（或更确切地说，各参数的名称）首先出现，而函数体在最后。所有参数都写成字符串形式。
当我们接收的是个函数字符串时，可以直接运行

示例

call & apply & bind

作用

call apply bind 改变的是函数运行时this的指向

用法

fn.call(context, ...args);当第一个参数为null、undefined的时候，默认指向window。参数用逗号隔开
fn.apply(context, args); 第二个参数是数组
const newFn = fn.bind(context, ...args); 不立即执行，返回一个this指向改变的函数

区别

手写

箭头函数和普通函数区别

  1. this指向，箭头函数不改变this指向，function为使用时所在对象
  2. 不能使用arguments对象
  3. 不能使用yield命令，所以不能用作Generator函数
  4. 不能使用new命令，因为没有自己的this，无法使用call和apply方法
    没有prototype属性，而 new 命令在执行时需要将构造函数的 prototype 赋值给新的对象的 proto
